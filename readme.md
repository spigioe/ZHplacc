# 🎓 platZH – Az egyetemi túlélőkészlet

[![Version](https://img.shields.io/badge/version-0.8.3--beta-blue)](https://github.com/felhasznalonev/platzh)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-PWA%20%7C%20Web-orange)](https://platzh.hu)

A **platZH** egy modern, reszponzív és felhasználóbarát webalkalmazás, amelyet kifejezetten egyetemi hallgatók számára lett tervezve. Segít rendszerezni az órarendet, nyomon követni a számonkéréseket és kezelni a napi teendőket, hogy soha ne maradj le semmiről. Meg már amúgy elegem van hogy nem tudom mi mikor van, az is volt a cél vele hogy erre megoldást találjak, csak kicsit túlnőtte magát a projekt.

## 🚀 Főbb funkciók

*   **📅 Intelligens Órarend:** Neptun-szinkronizáció (ICS) támogatás és egyéni órák rögzítése.
*   **✍️ Számonkérés Kezelő:** Közelgő ZH-k és vizsgák listázása visszaszámlálóval és helyszín adatokkal.
*   **📚 Tantárgy Nyilvántartás:** Kreditpontok, féléves statisztikák és követelmények egy helyen.
*   **✅ Teendők (Todo):** Napi fókusz segítő, hogy tudd, mi a legfontosabb feladatod aznap.
*   **🌓 Dinamikus Témák:** Teljes körű világos és sötét mód támogatás a szemkímélő éjszakai tanuláshoz.
*   **📱 PWA Támogatás:** Telepítsd mobilodra appként közvetlenül a böngészőből (iOS & Android).

## 🛠 Technikai háttér

Az alkalmazás **Single Page Application (SPA)** architektúrára épül, minimalista technológiai stacket használva a maximális sebesség és egyszerűség érdekében:

*   **Frontend:** [Bulma CSS Framework](https://bulma.io/) (modern Flexbox alapú design).
*   **Scripting:** Tiszta, moduláris **Vanilla JavaScript** (ES6+).
*   **Adattárolás:** Helyi perzisztencia és REST API integráció.
*   **Ikonok:** FontAwesome 6.

**Link az oldalhoz:** https://plat-zh.vercel.app/

## Telepítés mobilon:

### iOS (iPhone/iPad)
1. Nyisd meg az oldalt **Safari**-ban.
2. Koppints a **Megosztás** ikonra (négyzet felfelé nyíllal).
3. Válaszd a **"Hozzáadás a főképernyőhöz"** opciót.

### Android
1. Nyisd meg az oldalt **Chrome**-ban.
2. Koppints a jobb felső három pontra.
3. Válaszd a **"Telepítés"** vagy **"Webalkalmazás hozzáadása"** lehetőséget.
